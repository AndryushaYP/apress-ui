Загрузчик отложенных ресурсов

Модуль предоставляет глобальную функцию apressImport для загрузки скриптов и стилей. Функция принимает ссылку или массив ссылок на js-скрипты или css-файлы, и возвращает промис, который будет разрешен после загрузки всех скриптов и стилей.

Повторные вызовы apressImport с одинаковыми url не приведут к повторному запросу ресурсов (будет возвращен разрешенный промис).

Пример использования с одним ресурсом:

apressImport('path/to/script.js|style.css')
  .then(() => {
    // Код который должен выполниться после загрузки скрипта/стиля
  });

Пример использования с массивом ресурсов:

apressImport(['path/to/script_1.js', 'path/to/script_2.js', 'path/to/style_1.css', 'path/to/style_2.css'])
  .then(() => {
    // Код который должен выполниться после загрузки всех скриптов и стилей
  });
