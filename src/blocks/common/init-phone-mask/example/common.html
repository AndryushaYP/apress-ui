Модуль для инициализации телефонной маски

Подключение: import '@apress/apress-ui/dist/common/init-phone-mask';

Способы инициализации маски:

При инициализации модуля автоматически
обрабатываются все элементы с классом js-phone-mask.

Событие init:phoneMask на элементе document обработает
все элементы с классом js-phone-mask (аналогично инициализации).

Событие init:phoneMask на конкретном элементе ввода
обработает только этот элемент (не зависимо от класса).

Конфигурация маски:

Модуль читает следующие параметры из конфига app.config.initPhoneMask (при его наличии):

  • phoneCode - для обнаружения ввода по маске;
  • phonePattern - маска для номера;
  • phonePlaceholder - плэйсхолдер для поля ввода;

Дата атрибуты:

phoneMaskOptions - атрибут для конфигурации определенного инптута

Может содержать в себе следующие свойства:

  • not-remove-code - добавляет слушатель для бокировки удаления кода страны
  • only-default-code - добавляет предзаполненный код страны
